# Add "Done" button to All Clubs Paired screen & fix navigation flow

**Changes**

- **Green "All Clubs Paired!" screen**: Remove the auto-navigate timer. Add a white rounded rectangle button at the bottom with green text saying "Done".
- **When "Done" is pressed**: Navigate to the blue gradient "My Bag" screen showing your paired clubs and the "Edit Bag+" button.
- **Blue "My Bag" screen**: The "Done" button in the top right corner will navigate the user all the way back to the Profile screen (closing all intermediate screens).
